# ag2latin

Transliterate Ancient Greek text to Latin characters.

## Usage

```python
from ag2latin import ag2latin

ag2latin("μῆνιν")      # 'MĒNIN'
ag2latin("θεὰ")        # 'THEA'
ag2latin("ἑλώρια")     # 'HELŌRIA'
ag2latin("ῥίγος")      # 'RHIGOS'
```

## Command Line

```bash
python ag2latin.py "μῆνιν"
# μῆνιν -> MĒNIN
```
